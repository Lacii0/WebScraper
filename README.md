# 📚 Web Scraper Project

## 📖 Project Description
This project is a Python-based web scraper that automatically collects data from the **books.toscrape.com** website.  

The goal was to create a **scalable and automated solution** for extracting market data instead of relying on manual data entry.

---

## 💼 Business Value
As a business informatics student, I consider **data accessibility** essential.  

This tool:
- Automates data collection  
- Reduces human error  
- Structures raw web data into analyzable formats  
- Supports further data analysis and decision-making  

---

## ⚙️ Technologies Used
- **Python 3**
- **BeautifulSoup4** – HTML parsing  
- **Requests** – HTTP requests  
- **Pandas** – Data handling and export  

---

## 🚀 Installation & Usage

### 1. Clone the repository
``` bash
git clone https://github.com/Lacii0/WebScraper.git
cd WebScraper
```

### 2. Create a virtual environment
```
python -m venv .venv
.venv\Scripts\activate
```
### 3. Activate the virtual environment
```
pip install -r requirements.txt
```
### 4. Run the program
```
python scraper.py
```
